Features

Rotary Control

Browse and control music using a tactile rotary encoder.

Queue + Favourites

Instantly build playlists and access your favourite albums.

Minimal UI

Clean interface designed for small displays.

Hi-Fi Ready

Pair with a DAC HAT for audiophile sound quality.

Why This Exists

Modern music software is overloaded with features. This project explores a different idea: a simple physical interface that makes browsing and playing music enjoyable again.

Inspired by classic hi-fi components and tactile hardware interfaces.
