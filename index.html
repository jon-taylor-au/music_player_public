Design philosophy

The filesystem is the library

01
No metadata library
Your folder structure is your organisation. No database to build, no tags to fix before playback, no import process. Plug in a drive and play.
02
One control surface
A single rotary encoder does everything. Turn to browse. Press to play. Long press for settings. The interface disappears and the music takes over.
03
Appliance, not computer
Boots straight into the player. No desktop, no OS, no distractions. Closer in spirit to a CD player or network streamer than any software application.
04
State that remembers
Power it off, power it back on. The player resumes exactly where you left it — same position in the browser, same track loaded and ready.
05
Lossless first
FLAC, WAV, AIFF, ALAC — high-resolution formats are first-class citizens. Paired with a quality DAC the output is indistinguishable from dedicated audio hardware.
06
Open and hackable
Every behaviour is configurable in a single settings file. No hidden logic, no cloud dependency, no account required. It's yours to modify completely.
How to use it

One knob. Everything.

Rotary encoder
+ push button
  • Turn Browse folders and tracks up / down
  • Press Play or pause the selected track
  • Double press Jump back to the currently playing track in the browser
  • Long press Open the settings modal — volume, shuffle, repeat, gapless and more
  • Idle 60s Browser automatically returns to the now-playing track

Technical stack

Built on proven tools

Python Core application logic, event loop, UI orchestration, and all playback control.
Pygame Display rendering and keyboard/encoder input handling at 30 FPS.
VLC Audio engine via python-vlc. Handles all format decoding, gapless transitions, and volume control.
Mutagen Metadata extraction across all formats — ID3, Vorbis comments, and iTunes/MP4 tags.
Raspberry Pi Target hardware. Runs headless with a 7" display, rotary encoder, and external DAC.
DAC External digital-to-analogue converter for high-quality audio output, bypassing the Pi's onboard audio.
